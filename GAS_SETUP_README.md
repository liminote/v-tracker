# Google Apps Script 設定教學

為了讓網頁可以「寫入」資料到你的 Google 表單，我們需要設定一個簡單的後端腳本。

## 步驟 1：開啟腳本編輯器
1. 打開你的記帳 Google 試算表。
2. 點擊上方選單的 **「擴充功能」 (Extensions)** > **「Apps Script」**。

## 步驟 2：貼上程式碼
1. 刪除編輯器中原本的 `myFunction` 程式碼。
2. 打開專案資料夾中的 `google_apps_script.js` 檔案，複製全部內容。
3. 貼上到 Google Apps Script 編輯器中。
4. 按 `Ctrl + S` (或 `Cmd + S`) 儲存，專案名稱可以取名為「記帳後端」。

## 步驟 3：部署為網頁應用程式 (最重要！)
1. 點擊右上角的 **「部署」 (Deploy)** 藍色按鈕 > **「新增部署」 (New deployment)**。
2. 在左側齒輪圖示旁選擇 **「網頁應用程式」 (Web app)**。
3. 設定如下：
   - **說明**：記帳 API (可隨意填)
   - **執行身分 (Execute as)**：**「我」 (Me)** (這很重要！)
   - **誰可以存取 (Who has access)**：**「任何人」 (Anyone)** (這也非常重要，選這個才能讓你的網頁寫入資料)。
4. 點擊 **「部署」 (Deploy)**。
5. 第一次部署時，Google 會要求你 **「授權存取」**。
   - 點擊「授權存取」。
   - 選擇你的 Google 帳號。
   - 如果出現「Google 尚未驗證這個應用程式」的警示視窗：
     - 點擊左下角的 **「進階」 (Advanced)**。
     - 點擊最下方的 **「前往......(不安全)」 (Go to ... (unsafe))**。
     - 點擊 **「允許」 (Allow)**。

## 步驟 4：取得網址
1. 部署成功後，你會看到一個 **「網頁應用程式網址」 (Web App URL)** (以 `https://script.google.com/macros/s/.../exec` 結尾)。
2. **複製這個網址**。
3. 我們等一下會把這個網址貼到網頁的設定中。
