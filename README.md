# ∞ Vanny 的豐盛記錄

一個優雅的個人記帳分析網頁應用，直接整合 Google Sheets 資料，提供視覺化圖表分析和明細查詢功能。

## 功能特色

- 📊 **月份支出分析** - 長條圖顯示各月份支出趨勢
- 🎨 **分類支出分析** - 堆疊長條圖顯示各月份分類佔比
- 🔍 **智慧搜尋** - 支援分類、內容、專案等多維度搜尋
- 💰 **即時統計** - 顯示搜尋結果總筆數和總金額
- 🌈 **分類色彩** - 前8大分類自動配色，明細表格同步顯示
- 📱 **響應式設計** - 支援桌面和行動裝置

## 技術架構

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **圖表**: Chart.js
- **資料來源**: Google Sheets API v4
- **部署**: Vercel

## 使用方式

1. 取得 Google Sheets API 金鑰
2. 設定 Google Sheets 為公開或可連結存取
3. 在網頁中輸入 API 金鑰並儲存
4. 點擊「重新整理資料」同步最新資料

## 資料格式

Google Sheets 需包含以下欄位：
- 日期 (YYYY-MM-DD)
- 分類
- 內容
- 金額
- 專案
- 備註

## 配色方案

使用溫暖的米色系配色，營造舒適的使用體驗。
